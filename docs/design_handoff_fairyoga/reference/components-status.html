fair.yoga Status system · proposal

A class is in one of seven states

Activity states describe how full the class is — marked with a small earth-brown shape. Meta states describe a different condition altogether (drafted, cancelled, past) and are written as italic words. No second color, no chromatic noise — teal stays reserved for money.
