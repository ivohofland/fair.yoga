# Teacher app — UI kit (v2)

The one fair.yoga product: the teacher web app, rebuilt to the updated design brief. Mobile-first (420px column on cream), bottom tab bar, sand cards with the registration progress bar as the signature element.

- `index.html` — interactive prototype: tab bar navigation + screen switcher covering all lifecycle stages
- One `.jsx` per screen; all compose the shared components from `components/` via the project bundle
- `data.js` — seed data

Screens: Schedule (home), Class detail (adaptive by lifecycle: registering / today / completed), Create class (3 steps + confirmation), Class day, Post-class summary, Payment checklist, Students, Inbox, Tier selection (student side).
