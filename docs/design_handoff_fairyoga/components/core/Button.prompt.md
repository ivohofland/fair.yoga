Pill button, 48px tall, sans semibold 16, 24px horizontal padding. One primary per screen; full-width in mobile forms.

```jsx
<Button variant="primary" fullWidth>Create class</Button>
<Button variant="secondary">Share booking link</Button>
<Button variant="destructive">Cancel class</Button>
<Button variant="ghost">Try again</Button>
```

Hover/pressed use the defined teal steps; destructive is outline-only, never filled. Labels are sentence case, no exclamation marks.
