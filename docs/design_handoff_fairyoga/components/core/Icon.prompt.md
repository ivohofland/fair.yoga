Functional line icon (Lucide path data, stroke 1.75, currentColor, never filled). Use only where a word would be longer or less clear — the tab bar, card chevrons, checkmarks.

```jsx
<Icon name="calendar" size={24} />
```

Names: `calendar`, `users`, `inbox`, `settings`, `chevron-right`, `arrow-left`, `plus`, `check`, `x`, `share`. Inherits `color` from its parent. No decorative or colored icons.
