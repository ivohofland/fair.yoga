Pricing Algorithm — Test Cases

Ethical Yoga App — Income-based pricing engine

How the algorithm works

student price = (room cost + teacher rate) / sum of all ratios × student's ratio
Tier 1 — 0.5× Tier 2 — 0.75× Tier 3 — 1.0× Tier 4 — 1.25× Tier 5 — 1.5×
